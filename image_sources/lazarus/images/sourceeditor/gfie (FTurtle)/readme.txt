This folder contains source files for images used in IDE.
These source files created with Greenfish Icon Editor Pro.
Each file contains all sizes of particular image (or few images).
Filename suffix with digits in brackets means maximal magnification
of images containing in source. 

Sizes of images:
100% - 11x11
150% - 16x16
200% - 22x22
300% - 33x33

All files created by FTurtle from scratch.
Some images made for looking similar existing images from old Lazarus distribution
without any copying.

The source files for creating the  palette icons of the SqlDB components at 150% 
and 200% magnification can be found here.

The files are multi-layer gimp files for easier modification and changing
overlay icons. 

Each source file serves several components. Activate the layers needed for each 
component before exporting to png.


--------------------------------------------------------------------------------
Origin of images
--------------------------------------------------------------------------------
- Light blue cylinder: this is image "database_blue.png" taken from the FatCow
  icon set (http://www.fatcow.com/free-icons)
  
- The yellow lightning icon of  TSqlTransaction is the image "lightning.png"
  of the FatCow icon set (http://www.fatcow.com/free-icons).
  
- The yellow script icon if TSqlScript is the image "script_yellow.png" of the
  FatCow icon set (http://www.fatcow.com/free-icons).

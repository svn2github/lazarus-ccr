This folder contains the gimp source file for the images used internally by 
component spktoolbar.

The images used are taken the FatCow icon set (http://www.fatcow.com/free-icons, 
license Creative Commons Attribution 3.0).

The gimp source files contains all images needed for a given icon size on 
separate layers. Activate the layers needed for a given png image to be 
exported.
